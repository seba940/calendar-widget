# 📅 Google Calendar Grid Widget

구글 캘린더와 연동되어 바탕화면에서 일정을 직관적으로 확인하고 관리할 수 있는 파이썬 기반의 투명 달력 위젯입니다.

## 🚀 주요 기능

- **실시간 연동**: 구글 캘린더 API를 통해 일정을 자동으로 동기화합니다.
- **일정 관리**: 위젯 내에서 직접 일정을 추가, 수정, 삭제하고 완료 표시(✅)를 할 수 있습니다.
- **맞춤형 UI**: 테마(Black/White), 투명도, 글꼴 종류 및 크기를 자유롭게 설정할 수 있습니다.
- **음력 및 절기 표시**: 한국 음력 날짜와 24절기를 지원합니다.
- **위젯 고정**: 바탕화면 하단에 고정하여 다른 창에 방해받지 않고 사용할 수 있습니다.

## 🛠 설치 및 시작하기

### 1. 사전 요구 사항
- Python 3.8 이상이 설치되어 있어야 합니다.
- 필요한 라이브러리를 설치합니다:
  ```bash
  pip install google-api-python-client google-auth-httplib2 google-auth-oauthlib python-dotenv korean-lunar-calendar
  ```

### 2. 구글 API 설정
프로그램을 처음 실행하면 API 설정 가이드 창이 나타납니다.
1. [Google Cloud Console](https://console.cloud.google.com/)에 접속합니다.
2. 새 프로젝트를 생성하고 **Google Calendar API**를 활성화합니다.
3. '사용자 인증 정보' 메뉴에서 **OAuth 클라이언트 ID**를 생성합니다 (애플리케이션 유형: 데스크톱 앱).
4. 생성된 **Client ID**와 **Client Secret**을 프로그램의 설정 창에 입력하고 저장합니다.
5. 웹 브라우저가 열리면 구글 계정으로 로그인하여 권한을 승인합니다.

## 📌 부팅 시 자동 실행 설정 (Windows)

매번 직접 실행할 필요 없이, 윈도우 부팅 시 자동으로 프로그램이 켜지도록 설정할 수 있습니다.

1. `grid_calendar.pyw` 파일(또는 빌드된 `.exe` 파일)을 우클릭하여 **바로가기 만들기**를 선택합니다.
2. `Win + R` 키를 눌러 실행 창을 엽니다.
3. `shell:startup`을 입력하고 확인을 눌러 **시작프로그램** 폴더를 엽니다.
4. 생성한 바로가기 파일을 이 폴더로 이동시킵니다.

> **주의사항**: 프로그램 실행 파일과 설정 파일(`.env`, `token.json`, `settings.json`)은 반드시 같은 폴더 내에 유지되어야 합니다.

## ⌨️ 조작 방법

- **드래그**: 위젯의 빈 공간을 드래그하여 위치를 이동할 수 있습니다 (고정 모드 해제 시).
- **크기 조절**: 위젯 우측 하단 끝을 드래그하여 크기를 조절할 수 있습니다.
- **일정 클릭**: 날짜 셀이나 일정을 클릭하면 상세 보기 및 수정 창이 나타납니다.
- **드래그 앤 드롭**: 상세 보기 창에서 일정 왼쪽의 `≡` 아이콘을 드래그하여 일정 순서를 변경할 수 있습니다.
