# 📅 Google Calendar Grid Widget

구글 캘린더와 연동되어 바탕화면에서 일정을 직관적으로 확인하고 관리할 수 있는 파이썬 기반의 투명 달력 위젯입니다. 최근 업데이트를 통해 코드 구조가 모듈화되어 유지보수가 더욱 쉬워졌으며, 구글 인증 과정이 간소화되었습니다.

## 🚀 주요 기능

- **실시간 연동**: 구글 캘린더 API를 통해 일정을 자동으로 동기화합니다.
- **일정 관리**: 위젯 내에서 직접 일정을 추가, 수정, 삭제하고 완료 표시(✅)를 할 수 있습니다.
- **맞춤형 UI**: 테마(Black/White), 투명도, 글꼴 종류 및 크기를 자유롭게 설정할 수 있습니다.
- **음력 및 절기 표시**: 한국 음력 날짜와 24절기를 지원합니다.
- **위젯 고정**: 바탕화면 하단에 고정하여 다른 창에 방해받지 않고 사용할 수 있습니다.
- **모듈형 구조**: 대규모 코드를 기능별 모듈로 분리하여 안정성을 높였습니다.

## 🛠 설치 및 시작하기

### 1. 사전 요구 사항
- Python 3.8 이상이 설치되어 있어야 합니다.
- 필요한 라이브러리를 설치합니다:
  ```bash
  pip install -r requirements.txt
  ```

### 2. 구글 API 설정 (SSO 로그인)
프로그램을 실행하기 전 또는 실행 후 가이드에 따라 설정할 수 있습니다.

**방법 A: credentials.json 사용 (추천)**
1. [Google Cloud Console](https://console.cloud.google.com/)에서 **OAuth 클라이언트 ID**를 생성합니다 (애플리케이션 유형: 데스크톱 앱).
2. 생성된 JSON 파일을 다운로드하여 이름을 `credentials.json`으로 바꾼 뒤, 프로그램 폴더에 넣습니다.
3. 프로그램을 실행하면 자동으로 구글 로그인 창이 열립니다.

**방법 B: 직접 입력**
1. 프로그램 실행 후 나타나는 가이드 창에 **Client ID**와 **Client Secret**을 직접 입력합니다.
2. 웹 브라우저가 열리면 구글 계정으로 로그인하여 권한을 승인합니다.

## 📂 프로젝트 구조 (모듈화)

이 프로젝트는 유지보수 편의를 위해 다음과 같이 분리되어 있습니다:
- `grid_calendar.pyw`: 메인 실행 파일 및 UI 조정
- `google_calendar_api.py`: 구글 API 연동 및 데이터 처리
- `calendar_utils.py`: 음력 계산 및 공휴일 판별 유틸리티
- `config_manager.py`: 설정값 및 환경 변수 관리
- `ui_components.py`: 상세 창 및 가이드 UI 컴포넌트

## 📌 부팅 시 자동 실행 설정 (Windows)

1. `grid_calendar.pyw` 파일의 **바로가기**를 만듭니다.
2. `Win + R` -> `shell:startup` 입력 후 확인을 눌러 시작프로그램 폴더를 엽니다.
3. 생성한 바로가기 파일을 이 폴더로 이동시킵니다.

> **주의사항**: 실행에 필요한 파일들(`credentials.json`, `token.json`, `.env`, `settings.json`)은 반드시 실행 파일과 같은 폴더에 있어야 합니다.

## ⌨️ 조작 방법

- **드래그**: 위젯의 빈 공간을 드래그하여 위치 이동 (고정 해제 시).
- **크기 조절**: 우측 하단 끝을 드래그하여 크기 변경.
- **일정 클릭**: 상세 보기, 추가, 수정, 삭제 가능.
- **순서 변경**: 상세 창에서 `≡` 아이콘을 드래그하여 일정 순서 조정.
