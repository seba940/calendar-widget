# 📅 Google Calendar Grid Widget

구글 캘린더와 연동되어 바탕화면에서 일정을 직관적으로 확인하고 관리할 수 있는 파이썬 기반의 투명 달력 위젯입니다. 최근 업데이트를 통해 코드 구조가 모듈화되어 유지보수가 더욱 쉬워졌으며, 구글 인증 과정이 간소화되었습니다.

## 🚀 주요 기능

- **실시간 연동**: 구글 캘린더 API를 통해 일정을 자동으로 동기화합니다.
- **주간/월간 보기 전환**: 한 달 전체 보기와 한 주 단위 집중 보기 모드를 지원합니다.
- **일정 검색 및 목록**: 키워드를 통한 일정 검색과 다가오는 일정 목록(Agenda)을 한눈에 확인할 수 있습니다.
- **일정 관리**: 위젯 내에서 직접 일정을 추가, 수정, 삭제하고 완료 표시(✅)를 할 수 있습니다.
- **맞춤형 UI**: 테마(Black/White), 투명도, 글꼴 종류 및 크기를 자유롭게 설정할 수 있습니다.
- **시스템 트레이 및 시계**: 시스템 트레이 아이콘을 통해 백그라운드 실행을 지원하며, 위젯 내 실시간 시계 기능을 제공합니다.
- **음력 및 절기 표시**: 한국 음력 날짜와 24절기를 지원합니다.
- **구글 캘린더 색상 반영**: 구글 캘린더에서 설정한 일정별 색상을 위젯에서도 동일하게 확인할 수 있습니다.

## 📅 업데이트 내역 (Version History)

### v2.6.0 (2026-04-02)
- **일정 검색 기능**: 제목 및 메모 내용을 기반으로 과거와 미래의 모든 일정을 검색할 수 있는 기능을 추가했습니다.
- **주간 보기(Weekly View) 모드**: 월간 보기 외에도 한 주 단위로 일정을 더 상세히 볼 수 있는 주간 보기 모드를 구현했습니다.
- **일정 목록(Agenda) 보기**: 다가오는 일정을 리스트 형태로 모아보는 기능을 추가하여 효율적인 일정 관리가 가능해졌습니다.
- **헤더 UI 강화**: 오늘 이동, 뷰 전환, 검색 버튼을 헤더에 통합하여 접근성을 높였습니다.

### v2.5.0 (2026-04-02)
- **시스템 트레이(Tray) 지원**: 프로그램 종료 대신 시스템 트레이로 최소화하여 백그라운드에서 계속 실행될 수 있도록 기능을 추가했습니다.
- **실시간 시계 표시**: 위젯 상단에 현재 시간을 초 단위로 표시하는 실시간 시계 기능을 추가했습니다.
- **구글 캘린더 색상 동기화**: 구글 캘린더 웹/앱에서 설정한 일정별 고유 색상을 위젯에서도 그대로 반영하여 시각적 직관성을 높였습니다.
