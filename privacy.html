개인정보처리방침 (Privacy Policy)

Google Calendar Grid Widget(이하 '본 프로그램')은 이용자의 개인정보를 중요시하며, 관련 법령을 준수합니다. 본 방침은 구글 API 사용과 관련된 개인정보 처리 방식에 대해 설명합니다.

1. 수집하는 개인정보 항목 및 목적

본 프로그램은 이용자의 구글 계정에서 다음과 같은 정보를 활용합니다:

  • 구글 캘린더 데이터: 일정의 읽기, 쓰기, 수정 및 삭제를 위해 사용됩니다.
  • 목적: 바탕화면 위젯을 통해 이용자의 일정을 시각적으로 표시하고 관리 기능을 제공하기 위함입니다.

2. 데이터의 저장 및 공유

  • 로컬 저장: 구글 인증을 통해 획득한 액세스 토큰(token.json) 및 위젯 설정값(settings.json)은 오직 이용자의 로컬 컴퓨터 내 프로그램 실행 폴더에만 저장됩니다.
  • 제3자 제공: 본 프로그램은 수집된 데이터를 외부 서버로 전송하거나 제3자에게 제공하지 않습니다. 모든 데이터 처리는 이용자의 로컬 환경에서 이루어집니다.

3. 데이터 보호 및 보안

본 프로그램은 이용자의 비밀번호를 직접 수집하지 않으며, 구글의 표준 OAuth 2.0 인증 방식을 사용합니다. 인증 토큰은 이용자의 로컬 장치에 안전하게 보관됩니다.

4. 동의 철회 및 데이터 삭제

이용자는 언제든지 다음과 같은 방법으로 본 프로그램의 데이터 접근 권한을 철회할 수 있습니다:

  • 프로그램 폴더 내 token.json 파일을 삭제합니다.
  • 구글 계정 설정에서 본 프로그램의 권한을 제거합니다.
